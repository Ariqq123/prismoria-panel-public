CloudFlare Setup:
- CloudFlare Email: Your email for cloudflare profile
- CloudFlare Api Key: Your Global Api Key for your cloudflare profile
- Add your domain(s) to cloudflare

!! CAUTION !!
Don't forget to set IP alias to allocations in every node. IP Alias should be a domain which points to wings ip. Like node1.myhost.tld points to node1's ip. (A or CNAME record, DNS only)

- composer require cloudflare/sdk:dev-master
- yarn install
- yarn run build:production
- php artisan optimize
- php artisan migrate
